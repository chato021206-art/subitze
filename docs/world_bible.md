# SUBITZE Native — World Bible v1

**Sprint 0 成果物 / Last updated: 2026-04-13**

この文書は、SUBITZE ネイティブアプリ版（世界配信用）の **完全オリジナルの世界観・キャラクター設計書** である。現 Web 版 (stage.html) の既存世界観・キャラクターは一切参照せず、ゼロベースで設計した。既存 IP（Final Fantasy / Pokemon / Dragon Quest / Megami Tensei など）との類似、および地球上の宗教・神話・文化固有シンボルを全て排除している。

---

## 設計原則

1. **既存 IP 類似ゼロ** — キャラ名・シルエット・属性・スキル名いずれも既存ゲームの固有キャラを連想させない
2. **宗教・文化中立** — 十字・曼荼羅・鳥居・数珠・六芒星・八卦・カーバ・マンジ（卐/卍）・三つ目・輪廻など、特定の信仰に属するシンボル・概念を一切使用しない
3. **地球固有の神話・神名を避ける** — ギリシャ、ローマ、ヒンドゥー、仏教、神道、エジプト、北欧、ケルト、アボリジニ、マヤなど全ての現実の神話体系から命名・属性を借りない
4. **架空の造語中心** — 全キャラ名はオリジナルの造語・擬音・架空の音素で構成
5. **ビジュアルは抽象寄り** — 「特定の動物そっくり」ではなく「動物を抽象化した blob」を基本とし、法的類似性を避ける
6. **全年齢対象** — 武器・血・死・恐怖・苦痛の描写を避け、かわいさ・不思議さ・温かさ・神秘を基調とする

---

## 5 つの世界（Realms）

現 Web 版の 5 難易度（EASY → ULTIMATE）構造を維持しつつ、完全に新しい 5 つの架空の領域（Realms）を創造する。

### Realm 1: **Dewlight Glade** (ドゥライト グレード / 露光の園)
- **テーマ:** 夜明けの露が光を集める、穏やかな原っぱの領域
- **ムード:** 温かい・穏やか・初心者向け・癒し
- **パレット:** パステル緑、朝露の銀青、桃色、薄金
- **出題数帯:** 1〜9 個（EASY 相当）
- **世界観設定:** どこか遠くの世界、陽が昇る前の一瞬にだけ現れる秘密の原っぱ。露の粒が小さな生き物（ドラップ）となり、朝日が来るまで遊んでいる。

### Realm 2: **Chorus Grove** (コーラス グローヴ / 唱樹の森)
- **テーマ:** 葉がそれぞれ違う音を奏でる、歌う森
- **ムード:** 好奇心・遊び心・発見
- **パレット:** 深緑、琥珀、温かい茶色、クリーム
- **出題数帯:** 2〜9 個（NORMAL 相当）
- **世界観設定:** Dewlight Glade の奥にある森。風が吹くと葉が音階を奏で、住人たちはその響きに合わせて踊る。訪れるたびに違うメロディが聞こえる。

### Realm 3: **Kilnsands** (キルンサンズ / 窯野)
- **テーマ:** ガラスの砂粒でできた乾いた大地。住人は自分を少しずつ形作って暮らす
- **ムード:** 創作的・不思議・温かさ
- **パレット:** 琥珀、錆、クリーム、薄鉛色
- **出題数帯:** 10〜25 個（HARD 相当）
- **世界観設定:** 唱樹の森を越えると広がる、窯の名残のような砂漠。砂粒の一粒一粒がガラスで、住人はそれを集めて自分の体を作り直しながら歩く。

### Realm 4: **Tempest Reef** (テンペスト リーフ / 嵐礁群島)
- **テーマ:** 空に浮かぶ礁の群島。雲と雨と雷の住人が暮らす
- **ムード:** 勇壮・冒険・変幻自在
- **パレット:** 嵐の青、深い紫、銀、ミント
- **出題数帯:** 18〜45 個（EXTREME 相当）
- **世界観設定:** 窯野の上空に浮かぶ、珊瑚礁のような雲の島々。天気そのものが生き物となって暮らしている。風の強い日ほど彼らは元気になる。

### Realm 5: **Prism Hollow** (プリズム ホロウ / 虹洞)
- **テーマ:** あらゆる色と光が集まる、時間が流れない奇妙な空間
- **ムード:** 神秘・果てしない・畏れ
- **パレット:** 虹色／プリズム／白金／深いインディゴ
- **出題数帯:** 40〜99 個（ULTIMATE 相当）
- **世界観設定:** 4 つの世界を越えた先にある、すべての光が交わる空洞。ここでは時間が流れず、住人は光そのものとして存在する。最奥には「光の源」が眠っている。

### Realm 5: Prism Hollow
> すべての光が集まる、時間の流れない空洞。ここに来た光は、色を持ったまま永遠に漂い続ける。最奥には「光の源」が眠っていて、世界中を照らしている。

## 視覚設計ガイドライン（次スプリント以降への引き継ぎ）

### 全キャラ共通
- **基本形状:** 丸いぽてっとしたチビ型 blob（縦横 1:1〜1:1.2）
- **目:** 大きな anime スタイル、ハイライトで個性付け
- **口:** シンプルな `∪` または `:3` 形
- **体のディテール:** 過剰な武器・衣装・装飾なし
- **武器:** 持たせない（全年齢対象のため）

### 各 Realm のトーン
- **Dewlight Glade:** 柔らかいパステル、低彩度の可愛さ
- **Chorus Grove:** 中明度の暖色、音符や楽器モチーフを抽象化
- **Kilnsands:** 暖色＋ガラス質感、結晶のキラメキ
- **Tempest Reef:** コントラスト強め、雲・雷・雨のモチーフ
- **Prism Hollow:** 彩度最高、光り輝く、半透明

### 禁止事項
- 血・傷・暴力表現
- 特定の動物種をそのまま再現（例: 「完全にチョコボ」「完全にドラゴン」）
- 既存キャラの特徴的シルエット（モルボルの触手、マンドラゴラの葉頭、スライムの雫型など）の再現
- 宗教シンボル・神聖文字・特定文化の装飾
