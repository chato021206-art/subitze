# プロダクト仕様書: 全背景グラレコ風リデザイン

## 概要
SUBITZEの全背景（EASY/NORMAL/HARD/EXTREME/ULTIMATE）を、現在のFF14モチーフを活かしつつグラフィックレコーディング（グラレコ）風のスケッチスタイルに統一リデザインする。背景はマップ画面（ステージ選択）とゲーム画面の両方で共通使用される。

## コアコンセプト
- 現在の写実的SVG背景は美麗だが重厚すぎ、ゲームプレイ中の視認性を妨げうる
- グラレコ風にすることで、軽快・親しみやすい・視認性の高いビジュアルを実現
- 各難易度のFF14エリアテーマはアイコニックな要素で保持

## レイアウト制約（全背景共通）
背景SVGは `#game-bg`（`.map-bg`）として描画され、その上にUI要素が重なる:
- **マップ画面**: ステージノード（円形ボタン）が中央パスに沿って縦に並ぶ → 中央帯はシンプルに
- **ゲーム画面**: 上部にHPバー、中央にブロック表示エリア（`.game-stage-area`）、下部にキーボード（`#game-kb`、画面下33%）
- **重要**: ゲーム画面のブロック表示エリアとキーボードに視覚要素が被らないこと。描き込みは左右の余白と上部に集中させる

## 機能一覧

### F1: グラレコ共通スタイルシステム
- **概要**: 全背景で共通のグラレコ風ビジュアル言語を定義
- **ユーザーストーリー**: プレイヤーとして、全画面で統一感のあるスケッチ風デザインを体験したい
- **受け入れ条件**:
  - [ ] クリーム/オフホワイトの紙風ベース背景（`#F5F0E0`〜`#FFF8EC`）
  - [ ] 太めのダーク輪郭線（`#2A2A2A`、2-4px）で手描き風の揺らぎがある
  - [ ] グラデーション塗りなし、フラットな単色マーカー風塗り
  - [ ] 難易度ごとに微妙な紙色の違い（EASY=緑味、NORMAL=青味、HARD=琥珀味、EXTREME=珊瑚味、ULTIMATE=ラベンダー味）
  - [ ] ドゥードルアクセント（きらきら、波線、小さな記号）が散りばめられている
  - [ ] 描き込みはSVGの左右端と上部に集中し、中央〜下部はシンプルに保つ
- **優先度**: Must

### F2: 背景 EASY（グリダニア / 森）
- **概要**: 古代の森テーマをグラレコ風に
- **受け入れ条件**:
  - [ ] クリーム紙に緑味のベース
  - [ ] 巨大な木が太い幹 + 雲形キャノピーのスケッチ（左右配置）
  - [ ] 鳥居風の祠がシンプルなアウトライン（上部に配置）
  - [ ] エーテルグローがティール色のきらきらドゥードル
  - [ ] 野花・鳥のV字シルエットなどの小さなアクセント
  - [ ] 要素数25-35以下
- **優先度**: Must

### F3: 背景 NORMAL（リムサ・ロミンサ / 港）
- **概要**: 港町テーマをグラレコ風に
- **受け入れ条件**:
  - [ ] クリーム紙に青味のベース
  - [ ] 崖が左右に配置された太い角ポリゴン（フラットタン塗り）
  - [ ] 灯台が長方形 + 三角屋根 + 黄色い光の円（上部左右に）
  - [ ] 船が帆の三角形 + 船体カーブ（上部装飾として）
  - [ ] 海がフラットブルー + 手描き波線（最上部の背景として）
  - [ ] 要素数25-35以下
- **優先度**: Must

### F4: 背景 HARD（ウルダハ / 砂漠）
- **概要**: 砂漠都市テーマをグラレコ風に
- **受け入れ条件**:
  - [ ] クリーム紙に琥珀味のベース
  - [ ] メサが左右に配置されたフラットなバーントオレンジのポリゴン
  - [ ] ドームが半円アウトライン（ティール塗り、上部スカイラインに）
  - [ ] ミナレットが縦線 + 尖塔トップ（左右に配置）
  - [ ] 永遠の炎が涙形フレームアイコン
  - [ ] 要素数25-35以下
- **優先度**: Must

### F5: 背景 EXTREME（ラザハン / 南国）
- **概要**: 南国バザールテーマをグラレコ風に
- **受け入れ条件**:
  - [ ] クリーム紙に珊瑚味のベース
  - [ ] マーケット日除けがカラフルなフラット長方形（左右上部）
  - [ ] バンティングフラッグが手描き線上の小三角形（上部を横断）
  - [ ] ヤシの木が茶色カーブ幹 + 緑扇形（左右に配置）
  - [ ] 海がフラットターコイズ + 波線（最上部）
  - [ ] 要素数30-40以下
- **優先度**: Must

### F6: 背景 ULTIMATE（クリスタリウム / 水晶塔）
- **概要**: 水晶塔テーマをグラレコ風に（暗い夜景 → 明るいラベンダー紙）
- **受け入れ条件**:
  - [ ] ラベンダー色のクリーム紙ベース
  - [ ] 水晶塔がテーパード幾何形 + ファセット線 + 頂点スターバースト（上部中央）
  - [ ] 星が15-20個のアスタリスク型ドゥードル（上部散布）
  - [ ] 浮遊クリスタルが4-6個のダイヤモンド形（左右に散布）
  - [ ] 建物が基本長方形アウトライン（左右下部）
  - [ ] 要素数35-45以下
- **優先度**: Must

### F7: baseBgColors + CSS背景色更新
- **概要**: stage.html のフォールバック背景色とCSS背景色をグラレコの紙色に合わせる
- **受け入れ条件**:
  - [ ] `baseBgColors`（L1658付近）の全難易度をクリーム系紙色に更新
  - [ ] `.map-scroll[data-diff]`（L159-167）のCSS背景色を紙色に更新
  - [ ] `#game-view[data-diff]`（L326-335）のCSS背景色を紙色に更新
  - [ ] +バリアントも対応する紙色に
- **優先度**: Must

- ゴール: 砂漠と南国の背景がグラレコ風に統一

### Sprint 3: ULTIMATE 背景
- 対象機能: F6

## 非機能要件
- SVG要素数を現在の100+から25-45に削減（パフォーマンス向上）
- 生成パイプライン（Python → inline SVG → stage.html）は維持
- モバイルブラウザ（Safari/Chrome）で60fps描画
- ゲーム画面のブロック表示エリアとキーボード領域に背景描き込みが被らない
